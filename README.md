
# Library Management System SQL

This repository contains SQL scripts for a Library Management System (LMS).  
It demonstrates database creation, sample data insertion, and queries.

## Folder Structure

- `sql queriess/` → SQL queries for creating tables, inserting data, and retrieving info
- `docs/` → ER diagrams or screenshots (optional)

## Example Queries

1.Create table books,members,employees,issued_status,return_status 
2. Update an Existing table
3. Deleting the record from the list
4. Retrieve all books 

## Usage

1. Open your SQL client (MySQL, PostgreSQL, etc.)
2. Run the scripts in `sql queriess/` folder
